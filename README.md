# Car Gallery Simple Demo Project based on Jarvis Framework

## Tech Stack
- PHP 7.4
- Jarvis Framework (v1.0.1)
- Composer
- MySQL Database
- React.JS

## Run
```bash
docker-compose up -d --build
```

### And open http://localhost:3000 in your browser
## Stop
```bash
docker-compose down --rmi="local"
```
